# Desafio Conectar - Trabalhando com APIs REST em React

## Projeto
Aplicação frontend feita com React.js + Vite, para exibir informações do PIB brasileiro usando a API do IBGE.

## Tecnologias
- React.js
- Axios
- Recharts
- Styled Components 
- React Router 

## Rodar o projeto localmente

```bash
yarn install
yarn dev
